# 🚛 Sistema de Gestión de Flota - Empresa Hirata

## 📌 Descripción
Este proyecto corresponde al desarrollo de una solución informática para la empresa de transporte **Hirata**, enfocada en mejorar la gestión del mantenimiento preventivo de su flota de camiones.

El sistema permite registrar y monitorear el kilometraje de los vehículos, generar alertas de mantenimiento y administrar información relevante de la flota.

---

## 🎯 Objetivo
Automatizar el control del kilometraje y mantenimiento de los camiones para:
- Reducir fallas mecánicas inesperadas
- Disminuir costos operativos
- Mejorar la eficiencia en las entregas

---

## ⚙️ Funcionalidades Principales

- 🚚 Registro de kilometraje de camiones
- 📊 Gestión de información de vehículos y conductores
- 🔔 Alertas automáticas de mantenimiento (cada 5000 km)
- 🛠️ Gestión de mantenimientos (crear, leer, actualizar, eliminar)
- 🧪 Pruebas unitarias de los módulos del sistema

---

## 🗄️ Tecnologías Utilizadas

- Base de datos: MySQL
- Lenguaje: Java
- Otros: No aplica
